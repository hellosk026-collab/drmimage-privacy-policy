# DRM Image 4K Engine Privacy Policy

This repository is intended to publish the public privacy policy page for DRM Image 4K Engine.

## Publish Steps

1. Push this repository to GitHub.
2. Enable GitHub Pages from the `main` branch root.
3. Use the published HTTPS URL in Google Play Console.

## Expected URL

If this repository is named `drmimage-privacy-policy` under the `hellosk026-collab` account, the expected URL will be:

`https://hellosk026-collab.github.io/drmimage-privacy-policy/`
